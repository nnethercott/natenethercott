Stock Bot.

2020


  • Stock data is scraped from Yahoo finance using Python OOP and Selenium
  • –
  • Stock objects allow the user to retrieve history over a specified period and to export the data as a csv locally simply using the ticker symbol
  • –
  • Portfolio objects are initialized with stocks and allow for computation of simple metrics over the user's holdings (i.e. net gain/losses since inception)
  • –
  • Python's smtplib is used to add mailing features so users can automate weekly reports which summarize portfolio analytics
